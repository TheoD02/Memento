La ligne de commande

La commande 'LS' & Le déplacement entre dossiers

Afficher le manuel d'une commande (ici 'LS')

man ls

Afficher les fichiers du dossier courant

ls

Afficher les fichier & les fichiers caché du dossier courant

ls -a

Lister les fichiers du dossier courant

ls -l

Lister les fichiers & fichiers caché du dossier courant en version longue

ls -al

Afficher le chemin d'accès du dossier courant

PWD

Se deplacer dans les dossiers

cd

cd .. -> retour en arrière dans le dossier parent de celui actuel

cd Documents -> Pour aller dans le dossier Documents (doit être existant)

Afficher le manuel d'une commande (ici 'LS')

sudo -i (Path par defaut : ROOT) ou sudo -s (Path ou l'on est actuellement) -> Se login en tant qu'admin (su)

Mettre à jour la listes des paquets

sudo apt update

Mettre à jour les paquets

sudo apt upgrade

Trouver tous les paquets contenant le mot XML

apt search XML

Installer le paquet git

dpkg -i atom-amd64.deb

Installer le paquet git

dpkg -i atom-amd64.deb

Gestion de dossier & de fichiers

Créer un dossier (ici : website)

mkdir website

Créer un dossier & un sous-dossier en une ligne de commande

mkdir -p Memento/assets

Créer les trois dossier dans le dossier assets

mkdir css img js Pour créer un dossier avec espace utiliser un anti-slash pour echapper le caractère "\ " ou 'texte test'

Créer un fichier

touch style.css

Créer un fichier avec du texte dedans

cat 'texte' > index.html

Editier un fichier

nano index.html

Déplacer ou renomée un fichier

mv ./style.css ./assets/css -> deplace le fichier style.css dans ./assets/css
